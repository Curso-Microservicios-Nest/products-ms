## Description

Microservicios de Productos

## Instalación

```bash
npm install
```

## Ejecutar la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Migraciones

```sh
# Ejecutar las migraciones y crear la base de datos
npx prisma migrate dev

# Generar una nueva migración
npx prisma migrate dev --name MigrationName
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
